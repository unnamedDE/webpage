This datapack only works in minecraft java edition 1.14+!
When using this pack standalone it will only add money to the game. You may want to consider installing other datapacks.

The group system

Everything about the group system

At the start every player is in his own group. Group owners can invite the nearest player by running the command /trigger invite_player. The player invited has to accept the invite and his group ID will change to the other players group ID. Everyone except the group owner can leave with /trigger leave_group and the owner can kick the nearest player with /trigger kick_player.

The money system

Everything about the money system

At the start every player gets a preconfigured amount of money (default is 1000$). You can check your money with the command /trigger check_money.

You can send the nearest player money by running the command /trigger pay_money set <amount> and replacing <amount> with the amount you want to send.

The creative menu

Everything about the creative menu

You can view the creative menu by running /function unnamed_core:creative_menu and then select the datapack to view the contents of.

Click the number between the square brackets and you will get the corresponding amount of the item.

Reverse rotating

How to use reverse rotating

You can rotate the item in an item frame anti-clockwise by holding shift and right-clicking.
