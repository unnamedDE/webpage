Adding Recipes

Learn how to add recipes with NBT support

The easiest way to add recipes is to use my generator. Navigate to unnamedDE.tk/aclib-generator. There are two ways to use the generator, to use MCscript (a scripting language by Stevertus) or using vanilla MCfunction. MCscript will output code for one file whereas MCfunction will output code for four files

Using MCfunction

First choose a recipe ID which is unique (like "datapackname_recipename") and enter it in the input labeled "Recipe ID". In the input "Function path" you have to enter the "/function" path to where you want to place the MCfunction file (ex.: datapackname:path/to/function).

Now you are ready to input your recipe ingredients/outputs. Find the right slot and enter the item ID and if you want NBT. If the slot you are editing is an output you can choose the count as well.

When you are finished press "Generate" and it will generate four different codes for you. Create a file at the specified path ("Function Path") with the extension .mcfunction and input the contents of the left output field. Then create a folder with the name of the function file. Inside that folder you have to create three MCfunction files. 1.mcfunction, 2.mcfunction & 3.mcfunction and paste the contents from the output fields from left to right.

The last step is to create a file in your datapack at the path data/ac_lib/tags/functions/advanced_crafter_recipes.json and add the function path from above to the JSON list.

Using MCscript

First choose a recipe ID which is unique (like "datapackname_recipename") and enter it in the input labeled "Recipe ID". In the input "Function path" you have to enter the "/function" path to where you want to place the MCfunction file (ex.: datapackname:path/to/function). You also have to specify the path where MCscript will compile the output file.

Now you are ready to input your recipe ingredients/outputs. Find the right slot and enter the item ID and if you want NBT. If the slot you are editing is an output you can choose the count as well.

When you are finished press "Generate (MCScript)" and it will generate one text for you. Paste them to your .mcscript file and compile it. It will compile it into a file and a folder with three files. The path of the main function file should match with the path specified above.

The last step is to create a file in your datapack at the path data/ac_lib/tags/functions/advanced_crafter_recipes.json and add the function path from above to the JSON list.

You can save the recipe to edit later by pressing "export" and import it later by drag-and-drop.
